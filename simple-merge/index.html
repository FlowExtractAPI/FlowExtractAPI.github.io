Frequently asked

Does it change my original documents?

No. Your source files are only read. Merging always produces a new Google Doc — or, if you choose, appends to the document you already have open without removing anything that's there.

Are my documents stored anywhere?

No. Files are processed only to complete the merge you asked for, and no document content is retained. The only thing we keep is a small monthly counter of how many merges you've run, used to apply plan limits. Full details are in the privacy policy.

Which file types can I merge?

Google Docs, Microsoft Word (.docx and .doc), OpenDocument (.odt), Rich Text (.rtf), Markdown (.md), HTML, plain text (.txt) and PDF. Non-Google files are converted automatically. For PDFs, the text is recovered but the original page layout is not preserved.

Can I export the result as Word or PDF?

Yes. Tick the Save as PDF or Save as Word option and a copy is created alongside the merged Google Doc, with a one-click download.

What permissions does it ask for, and why?

It requests access to the documents you select or open (to read and merge them), the ability to create the merged file in your Drive, and permission to contact its own secure processing service. It does not request access to your entire Drive. See the privacy policy for the full list.
